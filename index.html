Where Are We In Space?

Welcome to my web app, "Where Are We In Space?". This application retrieves data from various API's, converts the data to a JSON format, and then displays the live statistics of the ISS along with who is currently on the spacecraft. The leaflet.js library API is also used to plot the real time position of the ISS over a map of our planet.
